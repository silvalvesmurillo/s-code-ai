# Roadmap de Estudos – GenAI e Prompt Engineering para Desenvolvedores de Software

Este roadmap é voltado para engenheiros de software que desejam iniciar do zero em IA Generativa e Prompt Engineering, com foco em aplicações práticas no desenvolvimento, revisão de código e geração de pequenos PRs.

---

## 1. Fundamentos de IA Generativa e LLMs

### O que estudar
- O que são Large Language Models (LLMs)
- Diferença entre IA tradicional vs IA generativa
- Conceito de tokens, contexto e limitações

### Leituras
- OpenAI – Introdução a modelos de linguagem  
  https://platform.openai.com/docs/introduction  

- Google – What are large language models?  
  https://cloud.google.com/learn/what-are-large-language-models
